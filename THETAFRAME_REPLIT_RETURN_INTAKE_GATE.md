# ThetaFrame Replit Return Intake Gate

Date: 2026-04-13

Use this gate before any Codex-side merge, transport, or normalization work begins.

## Artifact Integrity Checks
The returned Replit package is incomplete unless all of these exist:
- portable full artifact zip
- changed-file manifest
- build receipt
- screenshot pack
- screenshot manifest
- patch notes
- assumptions/deviations
- Codex handoff note
- receipts

If brand/loading assets were transformed, require:
- brand-asset notes

## Product Contract Checks
Do not move forward unless the returned artifact still preserves:
- Daily as the primary current-day execution lane
- Weekly as the weekly alignment/protection lane
- Vision as the longer-range continuity lane
- Life Ledger as the structured obligations/plans lane
- Baby KB as an admin-only feeder inside Life Ledger
- onboarding as orientation-only
- current route/module/admin/auth/permission semantics

## Reject Conditions
Stop and do not begin Codex conformance if the returned artifact:
- introduces a top-level Baby KB route
- shifts execution weight away from Daily
- makes calendar the system of record
- introduces silent assistant writes
- uses loading art for ordinary route changes
- breaks auth, onboarding, mode persistence, or module gating

## Next Action After Pass
Only after this intake gate passes should the next run begin for:
- artifact ingestion
- changed-file diff review
- transport into canonical code authority
- Replit scaffolding normalization
- final conformance receipts
