# UI Pre-Deploy Checklist

## Home / Entry
- Landing view loads without errors
- Navigation and primary actions visible
- Initial content legible and aligned

## Main Flows
- Primary workflow steps complete end-to-end
- Tab or view switching works as expected
- State persists where intended

## Forms & Interactions
- Inputs accept text and focus states show
- Buttons and toggles respond on click/tap
- Validation or empty-state messaging appears (if applicable)

## Mobile Layout
- Layout fits narrow screens without horizontal scroll
- Tap targets are reachable and sized appropriately
- Cards/sections stack cleanly with spacing

## Build & Assets
- Production build succeeds
- No missing icons or images
- Tailwind styles compiled (no @tailwind directives in output)
